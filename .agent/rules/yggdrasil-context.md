# Yggdrasil: durable cross-session memory

Yggdrasil gives you a durable, local-first memory of this project's decisions,
lessons and status — across sessions, tools and projects.

- **Before** non-trivial work, recall prior context with the `ygg_recall`
  (cross-project) or `ygg_bootstrap` / `ygg_search` (this project) MCP tools.
- **After** solving something reusable, save it with `ygg_remember`.

*No project memory has been injected here yet — use the MCP tools above for
manual queries. Enable auto-injection with `ygg hooks`.*
